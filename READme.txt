открыть проект с репозитория на PyCharm:
	> скачать .zip 
	> перенести из архива на рабочий стол или в любой файл
	> открыть проект в PyCharm

активация программы:
	> удалить [если есть]/установить venv [виртуальное пространство]
	> открыть file/setting -> project interpreter -> show all -> нажать на (+)
		>> нажать на поставить (+) -> django pillow, django-import-export
	> добавить конфигурацию: add configuration -> нажать на (+) и указать имя по необходимости
	> [clt+alt+R] в консоли прописать: makemigrations -> migrate -> runserver 

*после первого запуска нужно подождать пока все установится для работы
